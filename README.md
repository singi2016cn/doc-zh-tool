# doc-zh-tool

为 [php/doc-zh](https://github.com/php/doc-zh) 开发的工具集。

## 快速开始

初始化项目

```shell
curl -fsSL https://raw.githubusercontent.com/singi2016cn/doc-zh-tool/main/bin/init.sh | bash
```

## 可用的命令工具

```text
|__check.sh # 语法检查
|__create_file.sh # 创建新的翻译文件
|__en_file_hash.sh # 计算文件的 hash
|__init.sh # 初始化项目
```
